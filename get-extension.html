Add to Chrome

Install ZeroTrust Guardian in 5 simple steps.

📁
Step 1

Download the Extension

The extension files are in the frontend-extension/ folder of this project.

🧩
Step 2

Open Chrome Extensions

Paste this URL in your Chrome address bar and press Enter.

🔧
Step 3

Enable Developer Mode

Toggle the "Developer mode" switch in the top-right corner of the extensions page.

📦
Step 4

Load the Extension

Click "Load unpacked" and select the frontend-extension/ folder from this project.

🛡️
Step 5

You're Protected!

Browse to any website. The shield icon will appear showing your safety status in real time.

Requires Google Chrome · Backend must be running on localhost:8000
